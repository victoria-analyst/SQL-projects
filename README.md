# SQL-projects  
"Emails by Country" code creates a view that combines data about user accounts and email campaigns. It calculates daily metrics such as number of accounts, emails sent, opens, and visits, grouped by country and account settings.  
The final query returns the top 10 countries by account count or emails sent, helping analyze email campaign activity across countries.  

"Revenue and Cost" query generates a daily report by product category by combining data on sales, advertising costs, email campaigns, and user registrations. It calculates revenue, paid search costs, number of emails sent, open and click rates, and new registrations, and aggregates these metrics by date and category. The result helps evaluate sales performance and marketing effectiveness over time.
