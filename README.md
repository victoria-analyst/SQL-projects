# SQL-projects  
**"Emails by Country"** code creates a view that combines data about user accounts and email campaigns. It calculates **daily metrics** such as accounts number, emails sent, opens, and visits, grouped by country and account settings.  
The final query returns the **top 10 countries** by account count or emails sent, helping analyze email campaign activity across countries.  

**"Revenue and Cost"** query generates a daily report by product category by combining data on sales, advertising costs, email campaigns, and user registrations. The result helps **evaluate sales performance and marketing effectiveness** over time.  

**"Timestamp Details"** query extracts **date and time components** from the event timestamp.  
It converts the timestamp into separate fields: **year, month, day, hour, minute, and second**. This makes it easier to analyze **when events happened** and helps with time-based analysis of user activity.

**"Prediction Fulfillement"** query compares actual revenue with predicted revenue over time.  
It calculates **cumulative sums** for both values and shows the percentage of the **revenue target** achieved for each date.
